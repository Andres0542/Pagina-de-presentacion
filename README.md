# Pagina de presentacion

Mi pagina web de presentacion. El punto de este proyecto personal es mas que nada realizar una pagina interesante y unica. 

Esta primera version que he realizado busco mejorarla con mejores transiciones y un diseño que me caracterize.
Una vez que considere al proyecto terminado y listo para publicar, añadire el link al deploy para acceder desde la web.

# Ver la pagina
Para ver mi pagina web simplemente entras al archivo index.html

# Ejecutar el codigo
Si buscas ejecutar el proyecto, clona la rama main para la ultima version de mi pagina estable.
Una vez clonada, entras a la carpeta y dentro de estas abres un cmd o una terminal e instalas las dependencias:

```npm install```

Para trabajar con mis archivos de estilos SCSS, utilizaras el compilador de scss a css. Para mantenerlo funcionando 
automaticamente mientras trabajas, realiza:

```npm run dev```
